# Dental Maverick AI Brand Package

This folder contains a first-pass professional brand package for `Dental Maverick AI`, built around the cleaned geometric mark and the selected `Plus Jakarta Sans` wordmark.

## Included

- `index.html`
  - Visual brand guide with logo system, palette, typography, and example assets
- `assets/logo-primary-horizontal-dark.png`
  - Primary horizontal logo for light backgrounds
- `assets/logo-primary-horizontal-light.png`
  - Primary horizontal logo for dark backgrounds
- `assets/logo-secondary-stacked-dark.png`
  - Secondary stacked lockup for vertical placements
- `assets/logo-secondary-stacked-light.png`
  - Light stacked lockup
- `assets/logo-mark-dark.png`
  - Mark-only asset
- `assets/logo-mark-light.png`
  - Light mark-only asset
- `assets/icon-square-light.png`
  - Light square app/icon treatment
- `assets/icon-square-dark.png`
  - Dark square app/icon treatment
- `assets/favicon-512.png`
- `assets/favicon-192.png`

- `assets/business-card-front.png`

- `assets/letterhead.png`

## Brand Direction

- Style: modern-tech
- Tone: premium, precise, confident, digital-first
- Primary font: `Plus Jakarta Sans`
- Core palette:
  - `#1A1D2E` Maverick Ink
  - `#4B5568` Graphite
  - `#B9B4AA` Stone
  - `#E7E4DE` Cloud
  - `#F8F6F1` Porcelain
  - `#FFFFFF` White

## Rebuild Assets

Run:

```powershell
python "c:\devtest\dental\brand-package\build_brand_package.py"
```

## Notes

- The current package is optimized for digital usage and presentation-ready deliverables.
