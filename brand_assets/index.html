Usage Guidance

Do

  • Use the horizontal logo as the default primary signature.
  • Use the stacked logo where vertical space is more available.
  • Use the light version on dark backgrounds only.
  • Preserve clear space equal to the mark width around the lockup.

Avoid

  • Stretching or condensing the mark or wordmark.
  • Mixing other fonts into the main logo lockup.
  • Using low-contrast backgrounds behind the dark lockup.
  • Adding extra outlines, shadows, or glow effects.
